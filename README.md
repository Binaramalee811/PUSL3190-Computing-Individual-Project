# Final Year Project - Real-Time Traffic and Route Optimizer (PathFinder Mobile Application)

The project introduces a mobile application for Colombo District, Sri Lanka, to address traffic congestion, accident rates, and road conditions. The app uses historical data, real-time sensor inputs, and user-contributed reports to provide accurate, area-specific insights, empowering commuters with actionable insights and improved travel planning.
